🛠️ KubeJS Master Tool

O KubeJS Master Tool é um utilitário desenvolvido em Python para simplificar a criação de receitas customizadas (scripts KubeJS) para modpacks de Minecraft. O foco principal é automatizar a sintaxe complexa do Create Mod, garantindo que você gaste menos tempo digitando código e mais tempo desenvolvendo seu pack.
✨ Funcionalidades

O aplicativo oferece suporte para diversos tipos de receitas através de uma interface intuitiva:

    Shaped: Criação de receitas clássicas de bancada (Crafting Table).

    Fornalha: Gerador de receitas simples para fornalha.

    Create Mod Integration:

        Mixer: Inclui suporte para seleção de requisitos de calor (none, heated, superheated).

        Crusher: Controle de chance de output e suporte para outputs secundários.

        Deployer: Facilita a configuração de itens base e ferramentas.

        Sequenced Assembly: A parte mais robusta, permitindo configurar itens de transição, 
        quantidade de loops, sequência de máquinas, chance de falha e itens resultantes da falha.

    Remover: Atalho rápido para gerar scripts que removem receitas existentes.

🚀 Como Executar

Se você estiver usando a versão executável:

    Vá até a pasta dist.

    Execute o arquivo KubeJS_Master_Tool.exe.

Se preferir rodar via Python:

    Instale as dependências (Tkinter costuma ser nativo).

    Execute: python seu_arquivo.py.

    🛠️ KubeJS Master Tool

KubeJS Master Tool is a Python-based utility designed to streamline the creation of custom recipes (KubeJS scripts) for Minecraft modpacks. The primary focus is to automate the complex syntax of the Create Mod, ensuring you spend less time writing code and more time developing your pack.
✨ Features

The application supports several recipe types through an intuitive interface:

    Shaped: Traditional Crafting Table recipe generator.

    Smelting (Fornalha): Simple generator for furnace and smelting recipes.

    Create Mod Integration:

        Mixer: Full support for mixing recipes, including heat requirement selection (none, heated, superheated).

        Crusher: Control over output chances and support for secondary outputs.

        Deployer: Simplifies the configuration of base items and required tools.

        Sequenced Assembly: The most robust module, allowing configuration of transitional items, 
        loop counts, machine sequences, failure chances, and specific "junk" items for failed attempts.

    Recipe Remover: A dedicated tool to quickly generate scripts that disable existing recipes.

🚀 How to Run
Using the Executable:

    Navigate to the dist folder.

    Run KubeJS_Master_Tool.exe.

Running via Python:

    Ensure you have Python installed.

    Run: python your_script_name.py.
